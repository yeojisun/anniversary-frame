# 🎀 Y2K Photo Booth - 4th Anniversary 💿

뉴진스 공식 홈페이지(`newjeans.kr`) 감성의 **미니멀 Y2K 테마 4주년 기념 사진 프레임 웹 페이지**입니다.
다양한 레트로 프레임 테마 선택, 회전하는 CD 플레이어 위젯, 스티커 꾸미기 및 고해상도 캔버스 이미지 합성 다운로드 기능을 제공합니다.

---

## 🎨 주요 특징

- **뉴진스 Y2K 감성 디자인:** 모노톤 격자 레이아웃, 얇은 테두리(1.5px solid), 회전하는 CD 디스크, 파스텔 네온 포인트 컬러 및 90년대 테크 UI 디자인.
- **반응형 세로 해상도 레이아웃:** 뷰포트 높이(`vh`) 기반으로 크기가 반응형으로 비례 조절되어 모바일/태블릿 화면에서도 세로 스크롤 없이 한눈에 조작 가능.
- **노션 폰트 자산 적용:** 
  - **Genty:** 타이틀에 사용된 동글동글하고 귀여운 벌룬 스타일 폰트.
  - **Alagard:** 카운트다운 및 프레임 날짜에 사용된 복고풍 픽셀 폰트.
  - **Helvetica / Inter:** UI 텍스트 및 말풍선 스티커의 정갈하고 깔끔한 본문 폰트.
- **고화질 캔버스 합성 엔진:** 스티커 위치, 회전 각도, 크기 및 Genty 타이틀 텍스트 레이아웃을 그대로 1:1 병합하여 인쇄 가능한 고화질 이미지(`.png`)로 다운로드.

---

## 🚀 시작하기

프로젝트 폴더 내에 별도의 빌드 과정 없이 `index.html` 파일을 더블 클릭하거나 로컬 웹 서버를 구동하여 즉시 실행 가능합니다.

```bash
# 로컬 웹 서버 구동 (예시: http-server 사용 시)
npx http-server -p 8000
```

웹 브라우저에서 `http://localhost:8000`으로 접속하여 테스트할 수 있습니다.
